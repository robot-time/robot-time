# Hi, I'm Miles 👋

I build things that live at the intersection of AI, hardware, and human experience.

## What I'm Working On

### 🧠 AI & Machine Learning
- [**Miso Mini**](https://github.com/robot-time/Miso-Mini) — Train Phi-4-mini to think: draft → critique → refine → debate → compress. Synthetic data generation, QLoRA fine-tuning with Unsloth, and a full curriculum training pipeline.
- [**Microwave**](https://github.com/robot-time/Microwave) — Decentralized AI inference network. Anyone can run an expert node; a router picks the best specialists for each prompt and queries them in parallel. BitTorrent for AI, starting from a LAN prototype.
- [**decentralized-moe**](https://github.com/robot-time/decentralized-moe) — Desktop MoE (Mixture of Experts) network app with pywebview UI, auto-discovery, routing transparency dashboard, and an Ollama-style interface.
- [**NightAtlas AI**](https://github.com/robot-time/nightatlasai) — Educational AI chat app with PDF uploads, flashcard generation, and grading rubric support. Built with Flask and Firebase, integrated with Gemini API.

### 🌐 Browsers & Web
- [**Jeev**](https://github.com/robot-time/Jeev) — A minimal Electron browser with a sidebar, spaces, Chrome extension support, a command bar, and a Brave-powered new-tab search engine.
- [**NightAtlas**](https://github.com/robot-time/NightAtlas) — Browser extension that fixes a particularly aneurysm-inducing school UI (Daymap). Complete CSS/JS overhaul.

### 📱 Mobile Apps
- [**ApPeer**](https://github.com/robot-time/ApPeer) — iOS social networking app built with SwiftUI and Firebase.
- [**Capsul**](https://github.com/robot-time/Capsul) — Real-time iOS messaging app with SwiftUI and Firebase.
- [**Jot**](https://github.com/robot-time/Jot---The-Voice-Note-Taking-Device) *(private)* — Voice-driven productivity device for Raspberry Pi Zero W with a 2.13" e-ink display. Speak a thought, it transcribes and organizes. Includes a Mac companion app.

### 🏥 Health & Wellness
- [**Wellness**](https://github.com/robot-time/Health-App) — Next.js health tracking dashboard with Supabase, Recharts, and AI-powered insights via the Anthropic API.

### 🔌 Hardware & IoT
- [**MorseCode ESP32**](https://github.com/robot-time/morsecode-esp32) — ESP32-based Morse code device with WiFi, MQTT over SSL, LCD display, and OTA firmware updates. Physical buttons for dot/dash/space/backspace.

### 🎲 Fun Projects
- [**ABHS Rug Pool**](https://github.com/robot-time/ABHSRugPool) — Meme coin tracking website. Fake crypto pools with live profit/loss, user balances, and leaderboard.
